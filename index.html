Sobre Mim

Sou apaixonada por tecnologia, design e experiências digitais que encantam. Tenho experiência no desenvolvimento de interfaces modernas, responsivas e funcionais aliando criatividade e código para transformar ideias em soluções visuais elegantes e eficientes.

Meu foco é criar projetos com impacto real, combinando usabilidade, estética e performance. A cada linha de código, busco entregar valor, clareza e propósito.

Instituto Somos Aldeia

Projeto voluntário de site institucional desenvolvido em HTML, CSS e JavaScript, com o objetivo de oferecer uma presença
digital simples e funcional para a Associação Instituto Somos Aldeia um grupo de apoio a mães atípicas de Torres/RS.

Tecnologias e funcionalidades:

  • Site estático: HTML, CSS e JavaScript puro
  • Navegação intuitiva com páginas: Home, Sobre, Contato
  • Design responsivo e leve, adaptado para dispositivos móveis
  • Formulário de contato funcional, permitindo que visitantes
    entrem em contato diretamente com a organização

Diferenciais:

  • Hospedado gratuitamente na Vercel
  • Projeto realizado de forma voluntária
  • Estrutura pensada para futuras expansões
    (como blog, eventos, área para doações)

Thrive Digital MKT

Projeto profissional de site institucional desenvolvido em WordPress, com foco em presença digital estratégica para uma
agência de marketing. O objetivo foi criar uma plataforma visualmente moderna, responsiva e orientada à conversão de leads.

Tecnologias e funcionalidades:

  • WordPress com Elementor Pro
  • Design responsivo, leve e orientado à performance
  • Integrações com plugins de SEO, formulário e WhatsApp
  • Seções personalizadas para serviços, contato e CTA (call to action)

Diferenciais:

  • Layout moderno e profissional, com foco em conversão
  • Construído com blocos personalizados no Elementor Pro
  • Hospedado com ótima performance de carregamento

Inove Building Intelligence

Projeto profissional desenvolvido a partir de um design fornecido no Figma da empresa. A implementação foi feita manualmente utilizando apenas HTML, CSS e JavaScript, com foco total em fidelidade visual, performance e responsividade.

Tecnologias e funcionalidades:

  • Implementação em HTML, CSS e JavaScript puro
  • Formulário de contato funcional com validação
  • Design responsivo, com adaptação para desktop e mobile
  • Layout 100% baseado no design do Figma fornecido pela empresa
  • Seções dedicadas para serviços, diferenciais e portfólio de clientes

Diferenciais:

  • Desenvolvimento estruturado sem frameworks
  • Foco em usabilidade, leveza e carregamento rápido
  • Execução 100% fiel ao layout proposto pela equipe de design
